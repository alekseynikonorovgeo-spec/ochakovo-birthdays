# 🎂 Дни рождения сотрудников — ПК Очаково

Веб-приложение (PWA) со списком дней рождения сотрудников комплекса. Работает офлайн, устанавливается на телефон как обычное приложение.

## Возможности

- **Список всех сотрудников** с датами рождения, должностью, отделом и телефоном
- **Фильтр по периоду:** сегодня / неделя / месяц / все
- **Поиск** по ФИО, отделу, должности и телефону
- **Фильтры:** по отделам, по должностям, избранные люди
- **Избранные** показываются всегда — в любой вкладке периода
- **Звонок по тапу** на номер телефона
- **Возраст:** показывает текущий возраст и сколько исполнится
- Работает **офлайн** (после первого открытия)

## Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub (например `ochakovo-birthdays`)
2. Загрузите в него все файлы из этой папки:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`, `icon-512.png`, `icon-180.png`
3. В репозитории откройте **Settings → Pages**
4. В разделе **Build and deployment** → Source выберите **Deploy from a branch**
5. Branch: `main`, папка `/ (root)` → **Save**
6. Через 1–2 минуты приложение будет доступно по адресу:
   `https://ВАШ_ЛОГИН.github.io/ochakovo-birthdays/`

## Как установить на телефон

- **iPhone (Safari):** откройте ссылку → кнопка «Поделиться» → «На экран «Домой»»
- **Android (Chrome):** откройте ссылку → меню (⋮) → «Установить приложение» / «Добавить на главный экран»

## Как обновить список сотрудников

Данные хранятся прямо в `index.html` в массиве `BIRTHDAYS_DATA`. Формат строки:

```js
["Фамилия Имя Отчество", "Отдел", "Должность", "Телефон", "ГГГГ-ММ-ДД"]
```

Отредактируйте массив, сохраните файл, загрузите на GitHub — приложение обновится автоматически (появится зелёный баннер «Обновить»).
